# frontend-design

**Frontend design skill for Claude Code.**

Teaches Claude Code production-grade UI design for web and mobile — design system thinking, visual hierarchy, distinctive aesthetics, and platform-aware implementation.

## Install

```bash
# Requires ac marketplace
/plugin install frontend-design@ac
```

## What It Provides

A single skill (`frontend-design`) that loads when you work with UI/UX. Two modes:

- **Web**: HTML/CSS/JS, React, Vue, Next.js — responsive layouts, CSS animations, browser-optimized design systems
- **Mobile**: Flutter, React Native, SwiftUI — touch targets, safe areas, platform depth, native patterns

## Key Features

- Design process: bold aesthetic direction before coding (Purpose → Tone → Constraints → Differentiation)
- Complete design systems: spacing scales, type scales, shadow elevation, HSL color palettes
- Visual hierarchy: primary/secondary/tertiary levels, button hierarchy, label strategies
- Anti-AI-slop: no generic fonts, no clichéd palettes, no cookie-cutter layouts
- Mobile-aware: touch targets, safe areas, platform-specific depth (iOS blur vs Material elevation)
- Progressive disclosure: core skill + reference files for deep-dive topics
